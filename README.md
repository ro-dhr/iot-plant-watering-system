🎯 Objective: Build an IoT-based irrigation system that automatically waters a plant based on soil moisture while allowing remote monitoring and control from a mobile app.

🛠️ What was used: 
- ESP32 microcontroller
- Capacitive soil moisture sensor 
- DHT11 temperature/humidity sensor
- Relay module
- 5V water pump
- Arduino IDE
- Blynk IoT platform.

💡 What was learned: Gained hands-on experience with sensor integration, relay-driven actuation, IoT cloud dashboards, and troubleshooting hardware mismatches like logic-level voltage differences (3.3V vs 5V).
